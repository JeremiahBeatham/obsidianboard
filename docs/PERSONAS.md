# TabulaRasa — Personas

## Obsidian Mobile Sketcher
**Archetype:** The Vault Purist

### Demographics
| Field | Value |
|---|---|
| Device | Mobile (iPhone/iPad) |
| Tech comfort | Medium-High |

### Goals
1. Sketch inside a note without switching apps
2. Keep sketches re-editable, not flattened images

### Frustrations
1. Other Obsidian sketch plugins dump PNGs into the vault, cluttering it
2. Drawing tools that don't feel native to touch/Pencil input

### Behaviors
- Currently screenshots Notes app sketches and pastes as images (loses re-editability)

### Scenario
Jotting a quick diagram mid-note on an iPad with Apple Pencil, expecting it to feel like drawing in the native Notes app — then being able to come back and edit the same sketch later.

### Quote
> "I just want to draw in my note, not export-import an image every time I want to fix a line."

### Design Implications
- Re-editability and inline embedding are non-negotiable, not nice-to-haves
- Touch/Pencil responsiveness has to feel native, or the plugin loses its reason to exist
